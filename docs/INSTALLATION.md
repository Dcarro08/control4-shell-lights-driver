# Installation Instructions for Control4 Driver

## Prerequisites
Before starting the installation, make sure you have the following:
- A working Control4 system
- Necessary permissions to install drivers
- Basic knowledge of navigating the Control4 interface 

## Step-by-Step Installation

1. **Download the Driver**  
   Go to the [Control4 Shell Lights Driver](https://github.com/Dcarro08/control4-shell-lights-driver) GitHub repository and click on the green "Code" button, then choose "Download ZIP".  
   Alternatively, clone the repository using Git:
   ```bash
   git clone https://github.com/Dcarro08/control4-shell-lights-driver.git
   ```

2. **Extract the Files**  
   If you downloaded the ZIP file, locate it in your downloads and extract it to a preferred location.

3. **Import the Driver into Control4**  
   - Open your Control4 Composer Pro.
   - Navigate to the "Drivers" section.
   - Click on "Add Driver" and then "Import Driver".
   - Browse to the extracted driver files and select the appropriate `.driver` file.

4. **Configure the Driver**  
   - After the driver is added, navigate to the driver settings in Composer Pro.
   - Fill in the required fields such as IP address, username, and password for the lights you want to control.
   - Save the changes.

5. **Test the Installation**  
   - Use the Control4 interface to test the driver functionality.
   - Ensure that all lights respond as expected. Adjust settings as necessary.

## Troubleshooting
If you encounter any issues during installation, consider the following steps:
- Ensure that all prerequisites are met.
- Check the network connection between the Control4 system and the lights.
- Restart the Control4 system and try reinstalling the driver.

For further assistance, visit the [Control4 Community](https://community.control4.com) or consult the official Control4 documentation.

## Conclusion
Now you should have the Control4 Shell Lights Driver installed and ready to use. If you have any feedback or questions, feel free to reach out!
